# 02_Codigos

Contiene los scripts utilizados para la ejecución operacional del modelo,
desde la transformación de la lluvia, la simulación, escribir resultados y actualizar 
estados y series simuladas.

-**01_Crones**: Codigo escrito para ser ejecutado mediante cronetab.
-**02_Rain**: Codigo escrito para el analisis de la lluvia.
-**03_Ejec**: Codigo escrito para la ejecución del modelo como tal.
-**04_Graphics**: Código escrito para la realización de figuras y gráficas.
